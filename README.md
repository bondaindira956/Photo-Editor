Photo Editor

A simple and interactive web-based photo editor built using HTML, CSS, and JavaScript.
It allows users to upload images, apply filters such as brightness, contrast, saturation, and grayscale, and view the changes instantly.
This project demonstrates the use of JavaScript for image manipulation and provides an easy-to-use interface for basic photo editing tasks.

Features:

Upload and preview images

Apply filters (brightness, contrast, blur, grayscale, etc.)

Reset image to original

Responsive and user-friendly design


Tech Stack:
HTML | CSS | JavaScript
